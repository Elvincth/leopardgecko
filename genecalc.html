隱性基因 以白化作解説:

•白化 X 白化 = 100% 白化

•白化 X Het 白化 = 50% 白化，50% het 白化

•白化 X 普通 = 100% het 白化

•Het 白化 X Het 白化 = 25%白化, 75% 普通 而當中66％ het白化(註1) (2/3 有het, 1/3 沒有het) / = 25% 白化，50% 普通 het 白化，25% 普通沒有het 白化

共顯性基因 以雪花作解說

•超級雪花 X 超級雪花 = 100%超級雪花

•超級雪花 X 雪花 = 50%超級雪花，50％雪花

•超級雪花 X 普通 = 100%雪花

•雪花 X 雪花 = 25%超級雪花，50%雪花，25%普通

•雪花 X 普通 = 50%雪花，50%普通

顯性基因 以白黃作解說:

•2c 白黃 X 2c 白黃= 100% 2c白黃

•2c 白黃 X 白黃 = 50% 2c 白黃, 50% 白黃

•2c 白黃 X 普通 = 100% 白黃

•白黃 X 白黃 = 25% 2c 白黃, 50% 白黃 , 25% 普通

•白黃 X 普通= 50% 白黃
